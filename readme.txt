== Description ==

My Custom Login is the WordPress login plugin, allows site admin to add login/registration on their sites menu.
It uses the page wp-login.php for login and wp-signup.php for register, it displays the login/register link if you are logged out and logout link if you are logged in.


= Features =
* Your registration, log in, pages remains same no changes.
* Simple login registration links that allow users to login and register to your site.

== Installation ==

1. Upload the plugin to your 'wp-content/plugins' directory
1. Activate the plugin

== Frequently Asked Questions ==

= How do I change the look of forms? =

this plugin uses your wordpress sites login register pages by default so you dont need to any change in your forms.

= I can't get into my site! What do I do? =

Visit yoursite.com/wp-login.php.
